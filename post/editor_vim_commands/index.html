Command

  • エディタ起動

    • $ vim
      • ( $ view : Viewモードを使って、ファイルの書き換えミスの保存やアクシデントなどを防ぎたい時に便利！）

  • ビジュアルモード

    • v : 複数文字や行を選択できるモード

  • インサートモード

    • i: ファイルに書き込むためのモード

      • a : 次の文字からインサートモードに切替え
      • o : カーソルの下に空白を入れ、インサートモードに切替え
      • 0 : カーソル行に空白を入れ、インサートモードに切替え

  • コマンドモード

    • :

      • 保存 / 終了

        • :w : 上書き保存
        • :wq: 保存して終了
        • :q : 編集終了
        • :q!: 保存をせずに終了

      • 削除

        • :%d: 一括削除（ファイルの中身を空の状態にする）

      • ジャンプ（移動）

        • :${数字} : 指定した行数に移動
          • e.g.
          • :5 : 5行目に移動
          • :10 : 10行目に移動
        • :set number : 行数を表示
        • :set nonumber : 行数を非表示

      • コマンドを実行

        • :!{$コマンド} :
        • e.g.
          • :!ls : lsコマンド（一覧表示）
          • :!python3 test.py : Pythonファイルを実行
          • :!ruby test.rb : Rubyファイルを実行
        • :!! : 前のコマンドを実行

  • ノーマルモード

    • [esc]:

      • 基本移動（カーソル移動）

        • k: ↑ 上に移動
        • j: ↓ 下に移動
        • h: ← 左に移動
        • l: → 右に移動

      • 特殊移動

        • 0 : 行頭に移動
        • ^ / $ : インデントの先頭/行末に移動
        • { / } : ひとつ上/下の段落に移動
        • [[ / ]] : ひとつ上/下の空白行に移動
        • gg / G : ファイルの先頭/最後に移動
        • Ctrl + o: 移動前に戻る

      • 戻る(undo) & 進む(redo)

        • u : ひとつ前の状態に戻す
        • = : その行でインデント位置を自動修正
        • Ctrl + r : 直前の操作に進む

      • 削除

        • x : 1文字削除
        • dd : 1行削除
        • 2dd: 2行削除
        • 3dd: 3行削除
        • dw : 単語ごとに削除

      • コピー(ヤンク) & ペースト(プット)

        • yy :（ヤンク）1行コピー
          • {数字}yy:（ヤンク）複数行コピー
          • e.g.
            • 2yy: 2行コピー
            • 3yy: 3行コピー
        • p :（プット）カーソルの下にペースト
        • yyp: コピペ
        • ddp: カット＆ペースト
        • J : カーソル行と下の行を連結する

      • 検索 & 置換

        • \${ワード} : 検索
        • n : 次の検索結果に移動
        • N : 前の検索結果に移動
        • R : 置換

Convenience

  • 置換

    • 一括置換 
      # ファイル内 一括置換
:%s/{old}/{new}/g
      • :%s/${検索ワード}/${置換ワード}/g : 一括置換
      • :%s/${検索ワード}/${置換ワード}/gc: 確認しながら置換
      # 行内 一括置換
:s/{old}/{new}/g

      # 10~20行の内 一括置換
:10,20/{old}/{new}/g

      # [V] visual-modeで選択内 一括置換
:'<,'>s/{old}/{new}/g

  • コメントアウト

    • 一括コメントアウト
      1. Ctrl + v：
        • 矩形（くけい）選択モードで行をバッと選択
      2. Shift + i：
        • コメントアウト文字を入力（e.g. #、--、//…etc.）
      3. Esc　
        • 押下後にバッと入力されるっよ！
